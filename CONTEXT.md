# context.md — Planejamento do Projeto HistoriaDavizao

## Visão Geral

O Projeto HistoriaDavizao é um site conceitual desenvolvido em HTML + CSS como base para testes de utilização de Intelig Artificial, design visual moderno e storytelling digital.

O objetivo principal não é apenas criar uma página bonita, mas uma experiência visual impactante, com sensação de presença, identidade forte e narrativa envolvente.

O site será centrado na história de Davizão, personagem principal e figura lendária do projeto.

A proposta visual deve transmitir:

- impacto
- modernidade
- presença
- mistério
- sofisticação
- tecnologia
- storytelling cinematográfico

O projeto deve parecer premium, mesmo sendo simples academicamente.

---

## Stack Principal

### Tecnologias

- HTML5
- CSS3
- JavaScript (mínimo necessário)
- Motion.dev (para hero animations)
- Fancy Components (para textos animados)
- Git + GitHub
- Deploy futuro opcional (Vercel / Netlify / Render)

---

## Filosofia de Desenvolvimento

### Regras principais

- código simples
- visual forte
- CSS limpo
- pouca complexidade desnecessária
- responsividade real
- evitar excesso de `px`
- priorizar:
  - rem
  - em
  - %
  - vw
  - vh
  - clamp()
  - flexbox
  - grid

### Evitar

- CSS extremamente complexo
- posicionamento excessivo com absolute
- layouts quebrados
- valores fixos demais
- excesso de media queries desnecessárias

## Observação Importante para IA

Ao gerar código:

- manter simplicidade
- evitar overengineering
- priorizar clareza
- manter visual premium
- pensar como landing page de alto nível
- usar HTML + CSS bem estruturado
- responsividade obrigatória
- foco em UX e impacto visual

O projeto deve parecer grande,
mesmo sendo simples.
